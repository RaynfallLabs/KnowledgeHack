## Stats System
```
STR → Carrying capacity
CON → HP/SP pools
DEX → Armor Class
INT → Max spells
WIS → Quiz timer
PER → Sight radius
```

## Food Progression
```
Goblin → +CON (hardy)
Orc → +STR (strong)
Bat → +PER (echolocation)
Dwarf → +CON (endurance)
Gnome → +INT (magical)
Eye → +PER/WIS (awareness)
```
